Create, import, and manage Stride game engine packages (.stridepackage files).
Handles asset organization, namespace management, resource validation, and registry operations.

================================================================================
QUICK START
================================================================================

### Create a Package
```csharp
using HS.Stride.Packer.Core;

var settings = new ExportSettings
{
    LibraryPath = @"C:\MyStrideProject",
    OutputPath = @"C:\Output\MyLibrary.stridepackage",
    Manifest = new PackageManifest
    {
        Name = "MyLibrary",
        Version = "1.0.0",
        Description = "Custom Stride assets",
        Author = "Your Name",
        StrideVersion = "4.2+",
        Tags = new List<string> { "effects", "materials" }
    },
    SelectedAssetFolders = new List<string> { "Assets/MyAssets" }
};

var manager = new StridePackageManager(settings);
var packagePath = await manager.CreatePackageAsync();
```

### Import a Package
```csharp
var importSettings = new ImportSettings
{
    PackagePath = @"C:\MyLibrary.stridepackage",
    TargetProjectPath = @"C:\TargetProject",
    OverwriteExistingFiles = true
};

var manager = new StridePackageManager(new ExportSettings());
var result = await manager.ImportPackageAsync(importSettings);
Console.WriteLine($"Imported {result.TotalFilesImported} files");
```

================================================================================
PACKAGE FORMAT
================================================================================

File Extension: .stridepackage (ZIP archive)

Contents:
  manifest.json           # Package metadata + SHA-256 hash
  Assets/                 # All selected asset files (.sdprefab, .sdmat, etc.)
  Resources/              # All resource dependencies (textures, models, audio)
  [PackageName]/          # Code and platform-specific folders
  stridepackage.json      # Optional registry metadata (alongside .stridepackage)

================================================================================
SUPPORTED FILE TYPES
================================================================================

Resource Files:
  Images: .png .jpg .jpeg .tga .dds .bmp .gif .tiff .webp .hdr
  Models: .fbx .obj .dae .gltf .glb .3ds .blend
  Audio:  .wav .ogg .mp3 .flac .aiff
  Video:  .mp4 .avi .mov .wmv .webm
  Fonts:  .ttf .otf
  Data:   .json .xml .csv .txt .yaml .yml

Stride Assets:
  .sdprefab .sdm3d .sdmat .sdtex .sdscene .sdsnd .sdanim .sdskel

  .sdarch .sdphys .sdconvex .sdraw

COMMON PATTERNS
================================================================================

### Scan Project Before Export
```csharp
var manager = new StridePackageManager(new ExportSettings { LibraryPath = projectPath });
var validation = await manager.ValidateForExportAsync(projectPath);

if (!validation.IsValid)
{
    Console.WriteLine(validation.GetReport());
    return;
}

var scanResult = manager.ScanProject(projectPath);
Console.WriteLine($"Files: {scanResult.FileCount}, Size: {scanResult.TotalSize}");
```

### Search Registry
```csharp
var manager = new StridePackageManager(new ExportSettings());
var packages = await manager.SearchPackagesAsync("particle");

foreach (var pkg in packages)
    Console.WriteLine($"{pkg.Name} v{pkg.Version} - {pkg.Description}");
```

### Filter by Tags
```csharp
var effects = await manager.FilterPackagesByTagsAsync(new List<string> { "effects", "VFX" });
```

### Download and Cache
```csharp
var packages = await manager.GetRegistryPackagesAsync();
var pkg = packages.First(p => p.Name == "MyLibrary");

if (!manager.IsPackageCached(pkg))
{
    await manager.DownloadPackageToCacheAsync(pkg);
}

var result = await manager.InstallCachedPackageAsync(pkg, targetProjectPath);
```

### Cleanup Orphaned Resources
```csharp
var cleanup = new ProjectCleanupService(projectPath);
var analysis = await cleanup.AnalyzeProjectAsync();

Console.WriteLine($"Found {analysis.OrphanedResources.Count} orphaned files");
Console.WriteLine($"Total orphan size: {analysis.TotalOrphanedSize} bytes");

if (analysis.HasIssues)
{
    var result = await cleanup.DeleteOrphansAsync(analysis.OrphanedResources);
    Console.WriteLine($"Deleted {result.DeletedFiles.Count} files");
}
```

### Verify Package Integrity
```csharp
var isValid = await manager.VerifyPackageIntegrityAsync(packagePath);
if (!isValid)
    Console.WriteLine("Package corrupted or tampered with!");
```
